# ZeroSMM Project Rules

## Техстек проекта
- Python 3.8+
- OpenAI API (GPT-4o, DALL-E 3)
- VK API, Telegram Bot API
- Библиотеки: openai, vk_api, aiogram/telebot

## Стандарты кода
- Используй классы для организации функционала
- Всегда добавляй обработку ошибок в API-запросах
- Храни конфиденциальные ключи в config.py или .env файлах
- Добавляй docstrings к методам классов
- Используй типизацию где возможно

## Архитектурные паттерны
- Один класс = одна ответственность
- Генераторы в папке generators/
- Публишеры в папке social_publishers/
- Статистика в папке social_stats/

## Специфика проекта
- Методы генерации должны возвращать строки
- API-ключи передавай через конструктор класса
- Используй f-строки для промптов
- Тестируй каждый модуль через if __name__ == '__main__'
