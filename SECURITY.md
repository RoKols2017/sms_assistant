# Безопасность SMM-системы с ИИ

## 🔐 Рекомендации по безопасности

### 1. Управление секретными данными

**❌ НИКОГДА НЕ ДЕЛАЙТЕ:**
- Не добавляйте файл `.env` в Git
- Не коммитьте API ключи в код
- Не передавайте токены через URL параметры
- Не логируйте секретные данные

**✅ ПРАВИЛЬНО:**
- Используйте файл `.env` для локальной разработки
- Добавьте `.env` в `.gitignore`
- Используйте переменные окружения в продакшене
- Регулярно ротируйте API ключи

### 2. Структура файла .env

```bash
# OpenAI API
OPENAI_API_KEY=sk-your-actual-key-here

# VK API
VK_TOKEN=your-actual-vk-token-here
VK_GROUP_ID=123456789

# Telegram Bot API
TG_TOKEN=1234567890:your-actual-bot-token-here
TG_CHAT_ID=@your_channel_or_chat_id

# Дополнительные настройки
LOG_LEVEL=INFO
MAX_RETRIES=3
TIMEOUT=30
```

### 3. Проверка безопасности

Перед запуском системы выполните:

```bash
python test_env.py
```

Этот скрипт проверит:
- ✅ Наличие всех переменных окружения
- ✅ Корректность API ключей
- ✅ Доступность внешних сервисов
- ✅ Безопасность подключений

### 4. Логирование и мониторинг

**Что логируется:**
- ✅ Успешные операции
- ✅ Ошибки API
- ✅ Время выполнения
- ✅ Статистика использования

**Что НЕ логируется:**
- ❌ API ключи и токены
- ❌ Содержимое постов
- ❌ Персональные данные

### 5. Ограничения API

**OpenAI:**
- Лимит запросов: зависит от тарифа
- Максимальная длина промпта: 1000 символов
- Стоимость: ~$0.03 за изображение DALL-E 3

**VK API:**
- Лимит запросов: 3 запроса в секунду
- Требует токен с правами: wall, photos, groups
- Максимальный размер изображения: 5MB

**Telegram Bot API:**
- Лимит запросов: 30 сообщений в секунду
- Максимальный размер файла: 50MB
- Требует токен бота от @BotFather

### 6. Рекомендации по развертыванию

**Локальная разработка:**
```bash
# 1. Скопируйте .env.example в .env
cp .env.example .env

# 2. Заполните реальными данными
nano .env

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Проверьте конфигурацию
python test_env.py
```

**Продакшен:**
```bash
# Установите переменные окружения
export OPENAI_API_KEY="your-key"
export VK_TOKEN="your-token"
# ... остальные переменные

# Запустите систему
python test.py
```

### 7. Мониторинг и алерты

Настройте мониторинг для:
- 🔍 Превышения лимитов API
- 🔍 Ошибок аутентификации
- 🔍 Недоступности сервисов
- 🔍 Необычной активности

### 8. Резервное копирование

**Регулярно создавайте бэкапы:**
- Конфигурационные файлы
- Логи системы
- База данных (если используется)

### 9. Обновления безопасности

**Регулярно обновляйте:**
- Python зависимости
- API библиотеки
- Системные пакеты
- Мониторинг уязвимостей

### 10. Контроль доступа

**Ограничьте доступ к:**
- Файлу `.env`
- Логам системы
- API ключам
- Конфигурационным файлам

## 🚨 В случае компрометации

1. **Немедленно отзовите** все API ключи
2. **Смените пароли** всех аккаунтов
3. **Проверьте логи** на подозрительную активность
4. **Обновите** все секретные данные
5. **Уведомите** команду безопасности
